  • Code: https://github.com/kasimte/coursera-data-products-project

The App

The app (hosted here) uses the USArrests dataset included in R and uses three simple linear models to predict arrest rates based on urban population percentage.

You can change the urban population percentage using a slider and see the rates change (see screenshot below).

The code

The code can be seen completely on Github here. We can show partially what the app is doing by running some R code right here.

library(shiny)
library(datasets)
percent <- 72
murderModel <- lm(Murder ~ UrbanPop, data=USArrests)
predict(murderModel,data.frame(UrbanPop = c(72)))

##        1 
## 7.923238
